# AGENTS.md

## Commands

- Run tests: `npm test`
- Run the project check: `npm run check`
- Try the CLI locally: `node ./src/cli.js check .`

## Project Notes

- This is a small dependency-free Node.js CLI.
- Keep checks simple and explainable.
- Add or update tests when changing behavior in `src/doctor.js`.

## Workflow

- Prefer small changes.
- Verify changes with `npm test` before handoff.
- Keep README examples aligned with the CLI output.
